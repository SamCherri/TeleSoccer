# Reboot Baseline

Este repositório foi reiniciado com exclusão da base anterior e reconstrução mínima das camadas para novo ciclo.

## Mantido como princípio

- arquitetura limpa por camadas;
- frontend sem regra de negócio;
- API como dona do fluxo de jogo;
- PostgreSQL via Prisma;
- deploy em Railway.

## Próximo incremento

- engine de turnos canônica;
- persistência Prisma em repositório dedicado;
- cenas e painel de ação mobile-first.
