# TeleSoccer - Game Design

## 1. Visão do jogo

TeleSoccer é um jogo online de futebol via bot, focado em progressão de carreira, partidas jogáveis e expansão futura para um universo mais amplo do futebol.

---

## 2. Fantasia principal do jogador

O usuário deve sentir que:

- criou um atleta
- evoluiu esse atleta
- entrou no profissional
- participa de partidas
- influencia o resultado com suas escolhas
- interage com outros jogadores no modo multiplayer MVP
- joga algo com identidade própria

---

## 3. Loop principal

### 3.1 Loop de carreira
- criar jogador
- evoluir
- treinar
- tentar peneira
- entrar no profissional

### 3.2 Loop de partida
- iniciar partida
- receber contexto
- escolher ação
- receber resultado
- acompanhar eventos
- concluir a partida

### 3.3 Loop multiplayer MVP
- criar ou entrar em sessão
- disputar partida compartilhada
- acompanhar estado conjunto
- encerrar sessão

---

## 4. Sensação desejada

A experiência do TeleSoccer deve transmitir:

- progressão
- tensão de decisão
- identidade de futebol
- presença online
- senso de jogo, não apenas de sistema

---

## 5. Regras de simplicidade

Enquanto o MVP não estiver completo:

- priorizar clareza sobre complexidade
- priorizar experiência central sobre sistemas acessórios
- evitar features grandes que desviem do loop principal

---

## 6. Definição de MVP completo

O jogo só é percebido como MVP completo quando:

- carreira base estiver funcional
- partida jogável estiver funcional
- multiplayer mínimo estiver funcional
- visual mínimo estiver funcional
