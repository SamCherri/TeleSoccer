# TeleSoccer - MVP Completo Pré-Fase 3

## 1. Finalidade

Este documento define o que significa “MVP completo” no TeleSoccer antes da Fase 3.

A intenção é evitar que o projeto siga para fases mais avançadas sem antes fechar uma experiência mínima completa, coerente e apresentável.

---

## 2. O que é considerado MVP completo

O MVP completo do TeleSoccer deve entregar quatro pilares:

1. carreira base do jogador
2. partida jogável por turnos
3. multiplayer simples funcional
4. camada visual mínima funcional

---

## 3. O que precisa existir neste MVP

### 3.1 Carreira
- criação de jogador
- atributos iniciais
- treino semanal
- tentativa de peneira
- aprovação e entrada no profissional
- histórico básico

### 3.2 Partida
- início e fim de partida
- contexto do lance
- ações válidas
- timeout funcional
- placar
- eventos obrigatórios
- energia, lesão, cartões e suspensão
- persistência completa do fluxo da partida

### 3.3 Multiplayer
- criação de sala, convite ou mecanismo equivalente
- entrada de dois usuários humanos
- estado compartilhado
- progressão coerente da partida para todos os participantes
- regra mínima para ausência, abandono ou expiração

### 3.4 Visual
- identidade visual mínima
- cards visuais
- feedback visual de contexto
- feedback visual de gol, cartão, lesão, fim de partida e placar
- estrutura simples de assets

---

## 4. O que NÃO entra no MVP completo

- liga completa
- calendário de temporada completo
- mercado completo
- técnico usuário completo
- dono de clube
- escalação manual avançada
- gráficos avançados
- animações complexas
- modo espectador completo
- economia completa do ecossistema

---

## 5. Resultado esperado

Ao fim deste MVP, o usuário deve sentir que:

- existe um jogo online funcional
- existe progressão real do atleta
- existe partida jogável
- existe multiplayer mínimo real
- existe identidade visual mínima
- o projeto já parece um produto e não apenas um backend

---

## 6. Regra de decisão

Se uma funcionalidade não melhorar diretamente um dos quatro pilares do MVP completo, ela não deve ser priorizada antes da Fase 3.
